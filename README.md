# QuizAutomation

Quiz Automation Project Overview

This project demonstrates end-to-end Selenium automation testing for a dynamic quiz web application built using HTML, CSS, and JavaScript. The automation script validates the complete quiz workflow, ensuring the application performs correctly from launch to result display. Automated Features

Launch the quiz application

Start the quiz

Navigate through all quiz questions

Automatically select answers

Submit the completed quiz

Capture and display the final score

Close the browser after execution
